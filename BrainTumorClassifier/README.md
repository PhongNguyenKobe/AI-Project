## 🇬🇧 Introduction
Today we will build a **brain tumor classifier** in Python using **PyTorch**.  
The goal of this project is to demonstrate how machine learning can be applied to medical image classification.

⚠️ **Disclaimer**: This is **not medical advice**.  
This is purely a **programming tutorial** intended to teach machine learning concepts.
